# Glossary

The canonical language of this workspace. Lessons, notes, records, and reports use these terms and no
others.

## How this file works

- **A term is added only after it has been used correctly**, not when it is first introduced. This
  file is a record of compressed knowledge, not a dictionary to read ahead in. Compressing a concept
  into one tight sentence *is* the evidence that it is understood — so the student writes the
  definition, and the instructor sharpens it.
- **Definitions are one or two sentences.** Say what the term *is*, not how to do it.
- **Be opinionated.** When several words exist for one concept, pick one and list the rest under
  _Avoid_. This is how language compresses.
- **Use the glossary's own terms inside its definitions.** This is what makes later terms cheap.
- **Revise in place.** A definition written in week one may be wrong by week six. Fix it; do not
  leave a stale entry.

Format:

```md
**Term**:
One or two sentences. What it IS.
_Avoid_: alias, other alias
```
