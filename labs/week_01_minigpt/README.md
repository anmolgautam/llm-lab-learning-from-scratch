# Week 01 — MiniGPT Lab

Implementation space for the first lab: a minimal character-level decoder-only GPT
that can overfit one batch and generate text (Phase 2). Phase 1 (next-token prediction)
is the conceptual groundwork built in `lessons/lesson_01_next_token_prediction/`.

> `week_NN` is a sequence label, not a calendar week.

