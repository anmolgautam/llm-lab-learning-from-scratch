# Mission shifted from interview prep to LLM systems engineering mastery

Status: active

The AI runtime interview that caused Phase 8 (inference) to be front-loaded on 2026-06-25 is no
longer the goal. The mission is now to become a hireable LLM systems / post-training engineer, with
`TinySeek` as the evidence artifact. See [[MISSION.md]].

## Implications

- Teaching returns to strict dependency order at Phase 1. Nothing is skipped for exam speed.
- `lessons/interview_ai_runtime_inference/` is **previewed, not earned**: the material was written
  under time pressure and no check was ever passed against it. It stays as early exposure — it will
  make Phase 8 faster — but it grants no mastery credit and its content must be re-derived when
  Phase 8 arrives properly, against a model we actually built.
- Removing the deadline removes the external forcing function that would have tested retention for
  us. Retention is now an explicit system: mastery levels and a spaced review deck (`review/deck.md`).
