# Lesson 00 — Orientation

Goal: understand the lab map before building anything.

- **What we're building:** the core decoder-only LLM stack from scratch — see [`curriculum.md`](../../curriculum.md).
- **How we work:** the teaching method and standards live in [`instructor.md`](../../instructor.md).
- **Where things go:**
  - `lessons/` — learning space (explanations, notebooks, exercises, notes, checks).
  - `labs/` — implementation space (package code, scripts, tests, configs).
  - `reports/` — experiment writeups.
  - `runs/` — generated outputs (mostly git-ignored).
- **What "done" means:** explain it, implement it, checks pass, ran one experiment, wrote a Markdown summary, answered review questions, updated `progress.md`.

Progress is tracked by evidence, not calendar time. See [`progress.md`](../../progress.md).
